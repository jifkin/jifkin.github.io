Kuramoto Oscillators

I've used a class of coupled phase oscillators called Kuramoto oscillators to model neural activity at a global scale. With this model, I was able to demonstrate how two groups with distinct connectivity patterns and simulated dynamics appeared in a set of brain images from the HCP.

The code used for my grouping algorithm can be found here.
